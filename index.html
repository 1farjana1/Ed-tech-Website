Grid is made for two-dimensional layout while Flexbox is for one.This means Flexbox can work on either row or columns at a time, but Grids can work on both. Flexbox, gives you more flexibility while working on either element (row or column).

Tailwind CSS was initially developed by Adam Wathan, and the first version was released back on the 1st of November, 2017. Tailwind CSS is a utility-first CSS framework for building custom user interfaces rapidly and efficiently. It is an inline styling used to achieve a sleek interface without writing code for your own CSS. It is a highly customizable, low-level CSS framework, and it provides all the building blocks the developer needs to build a fantastic interface for any website.
Bootstrap was developed by Mark Otto and Jacob Thorton during an internal competition at Twitter in August 2011.It is an open-source framework that is used to design responsive websites better, faster, and easier. It is a beginner-friendly and the most popular open-source framework that includes HTML, CSS, and JavaScript.It is based on object-oriented CSS. Bootstrap helps to design and develop website templates quickly.

The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used when talking about design and layout.The box model in CSS is a set of rules that determine how your web page is rendered on the internet.

A semantic element clearly describes its meaning to both the browser and the developer.Examples of non-semantic elements: div and span- Tells nothing about its content. Examples of semantic elements: form , table and article - Clearly defines its content.
